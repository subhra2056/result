# Result Portal

Live site: [Open the portal](https://subhra2056.github.io/result/)

## Enable the live link

1. Push `index.html`, `images.png`, and this `README.md` to GitHub.
2. Open the GitHub repo: `https://github.com/subhra2056/result`
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, select:
   - Source: **Deploy from a branch**
   - Branch: **master**
   - Folder: **/(root)**
5. Click **Save**.

After a minute or two, the live site should open at:

`https://subhra2056.github.io/result/`
